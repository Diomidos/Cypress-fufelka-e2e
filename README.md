Установить npm install
Установить cypress 
Запустить тест командой npx cypress open

------------------------------------------------------------------------------------------------------------------------------------------------------
В тесте описан сценарий e2e теста. От авторизации до проверки смены аватара после перезагрузки страницы.

Проверка профиля после перезагрузки страницы выполняется через команду cy.reload().

Автоматически генерируются для теста, аватарки из файла cypress/fixtures/avatar_*.jpg
